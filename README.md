LocAuto é um sistema completo de gerenciamento de locação de veículos, construído com Java 21, Spring Boot 3.x e MySQL. Implementa Spring Security com múltiplos perfis de acesso (Cliente, Funcionário, Gerente) e oferece um Dashboard SPA (JS/HTML/CSS) para gerenciar o CRUD de Carros (com exclusão lógica) e o ciclo de vida dos Contratos (incluindo cálculo de valor e liberação automática de frota).
